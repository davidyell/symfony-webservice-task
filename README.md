# Symfony Webservice

Creation of a restful api for managing expenses.

## Tooling
* PHP Storm
* SQLite3
* Postman

## Scripts
* PHPCS `composer phpcs`
* OpenAPI `composer swagger`

## Enhancements
Using a tool like [Fractal](https://fractal.thephpleague.com/) to manage more complex json responses.
